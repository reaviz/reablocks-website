# Navigation
The Navigation component serves as the primary entry point for users to explore an applicationâ€™s key sections. It provides a clear, consistent structure for navigating between core pages or features, ensuring users can easily orient themselves and move through the interface. Typically positioned at the top or side of the layout, it may include icons, labels, dropdowns, or interactive states for active and hovered items.

## Quick Start
To start using the `NavigationBar` component, import it from the `reablocks` library along with `NavigationButton`.

<Runner code={`
  import { NavigationBar, NavigationButton } from 'reablocks';
  import { useState } from 'react';

  export default function App() {
    const [active, setActive] = useState<number>(1);

    return (
      <NavigationBar >
        <NavigationButton
          active={active === 1}
          onClick={() => setActive(1)}
        >
          Home
        </NavigationButton>
        <NavigationButton
          active={active === 2}
          onClick={() => setActive(2)}
        >
          Dashboard
        </NavigationButton>
        <NavigationButton
          active={active === 3}
          onClick={() => setActive(3)}
        >
          Settings
        </NavigationButton>
      </NavigationBar>
    );
  }
`} />
