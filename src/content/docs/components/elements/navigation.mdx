# Navigation
The `Navigation` component renders a themed list of links for primary or
secondary navigation. It was added in `10.0.0`.

## Quick Start
To start using the `Navigation` component, import it from `reablocks`.

<Runner code={`import { Navigation } from 'reablocks';

const items = [
  { label: 'Dashboard', href: '/' },
  { label: 'Analytics', href: '/analytics' },
  { label: 'Settings', href: '/settings' }
];

export default function App() {
  return <Navigation items={items} />;
}`}
/>

## Design and Usability Tips

- **Highlight Current Location:** Always indicate the active item so users
  know where they are in the application.
- **Keep Labels Short:** Aim for one or two words per item; long labels make
  scanning slow.
- **Pair with Icons Sparingly:** Icons reinforce meaning, but only when their
  metaphor is universally understood.
