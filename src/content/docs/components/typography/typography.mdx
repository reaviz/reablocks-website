# Typography
The typography components are semantic HTML wrappers that ship with sensible
defaults for every level of headings and body text. They're designed to be
composed and styled with Tailwind utility classes via `className` or via the
shared `typography` theme.

## Components

- `H1` – page title
- `H2` – primary section heading
- `H3` – sub-section heading
- `H4` – small heading
- `H5` – tertiary heading
- `H6` – smallest heading
- `P` – paragraph / body text
- `Lead` – emphasized intro paragraph
- `Large` – emphasized inline text
- `Small` – de-emphasized inline text
- `Muted` – muted helper text
- `BlockQuote` – block quotation

## Quick Start
To start using the typography components, import them from `reablocks`.

<Runner code={`import { H1, P, Muted } from 'reablocks';

export default function App() {
  return (
    <>
      <H1>Welcome</H1>
      <P>Build beautiful interfaces with reablocks.</P>
      <Muted>Last updated today.</Muted>
    </>
  );
}`}
/>

### Customizing with `className`
The old `color`, `variant`, `fontStyle`, and `disableMargins` props were
removed in `10.0.0`. Apply Tailwind utilities directly via `className`, or
override the shared theme to change defaults globally.

```tsx
import { H1, P } from 'reablocks';

<H1 className="text-primary tracking-tight">Title</H1>
<P className="font-bold italic">Bold italic body</P>
```

## Theme
Typography uses a per-element theme. Every key maps to a single string of
Tailwind classes that's applied to that semantic element.

<ThemeRender name="typography" />

```ts
interface TypographyTheme {
  h1: string;
  h2: string;
  h3: string;
  h4: string;
  h5: string;
  h6: string;
  p: string;
  blockquote: string;
  lead: string;
  large: string;
  small: string;
  muted: string;
}
```

Learn more about how to customize in the [Theme documentation](/docs/theme/getting-started).

## Migrating from 9.x

| Before | After |
| --- | --- |
| `PageTitle` | `H1` |
| `PrimaryHeading` | `H2` |
| `SecondaryHeading` | `H3` |
| `SmallHeading` | `H4` |
| `Sub` | `H5` |
| `<Text>` | `P`, `Small`, or `Muted` |
| `<Text fontStyle="bold">` | `<P className="font-bold" />` |
| `<Text color="primary">` | `<P className="text-primary" />` |
| `<Text variant="mono">` | `<P className="font-mono" />` |

See the full guide in [Migration](/docs/getting-started/migration#migrating-from-9x-to-10x).
